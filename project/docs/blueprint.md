# **App Name**: SpendWise

## Core Features:

- Quick Expense Logger: A streamlined interface to record daily expenses with amount, date, and description fields.
- Cloud Sync Storage: Securely store and persist all expense records in a Cloud Firestore database for real-time accessibility.
- AI Insight Tool: An intelligent categorization tool that uses reasoning to automatically assign categories to expenses based on the user's transaction descriptions.
- Summary Dashboards: Daily and weekly summary reports that provide a high-level overview of total spending patterns.
- Visual Category Breakdown: Interactive charts showing percentage spending across different custom categories.
- Export Deliverables: Download your expense logs as structured files for offline viewing or external accounting.

## Style Guidelines:

- The palette is based on 'Financial Trust', featuring a professional Royal Blue (#2D58B8) as the primary color for buttons and primary actions.
- The background uses a highly desaturated tint of the primary hue (#F7F8FA) for a clean, light financial-ledger feel.
- An accent Teal (#3388A3) is used for emphasis and trend indicators, providing distinct contrast against the primary blue.
- Uses 'Inter' (sans-serif) for the primary interface text to maintain a modern and objective look. 'Source Code Pro' (monospace) is used for all numeric figures and expense amounts to ensure columnar readability.
- A centered, card-based layout inspired by minimalist dashboard design, focusing the user's attention on entry and summaries without visual clutter.
- Clean, thin-line icons for category icons (e.g., dining, utilities, transport) to keep the UI lightweight.
- Soft vertical transitions for chart entries and smooth fading on submission feedback to reinforce a sense of stability.
